<p>There is no linear relationship between Hue and RGB. Hue is defined piecewise in 60Â° chunks (<a href="http://en.wikipedia.org/wiki/HSL_color_space#General_approach">http://en.wikipedia.org/wiki/HSL_color_space#General_approach</a>), and therefore there is not a simple matrix conversion between HSV and RGB.
To change the hue of an image, you can use the following method :</p>

<pre><code>public Bitmap changeHue( Bitmap source, double hue ) {
    Bitmap result = Bitmap.createBitmap( source.getWidth(), source.getHeight(), source.getConfig() );

    float[] hsv = new float[3];
    for( int x = 0; x &lt; source.getWidth(); x++ ) {
        for( int y = 0; y &lt; source.getHeight(); y++ ) {
            int c = source.getPixel( x, y );
            Color.colorToHSV( c, hsv );
            hsv[0] = (float) ((hsv[0] + 360 * hue) % 360);
            c = (Color.HSVToColor( hsv ) &amp; 0x00ffffff) | (c &amp; 0xff000000);
            result.setPixel( x, y, c );
        }
    }

    return result;
}
