<p>First thing you need to implement a <code>javax.swing.Timer</code>. Trying to call <code>Thread.sleep()</code> will block the <a href="http://docs.oracle.com/javase/tutorial/uiswing/concurrency/index.html" rel="nofollow">Event Dispatch Thread</a>. See <a href="http://docs.oracle.com/javase/tutorial/uiswing/misc/timer.html" rel="nofollow">How to use Swing Timers</a>. Here is the basic construct</p>

<pre><code>Timer(int delay, ActionListener listener) 
</code></pre>

<p>where <code>delay</code> is the time in milliseconds you want delayed and the <code>listener</code> will listen for the <code>Timer</code> <code>ActionEvent</code> fired every <code>delay</code> milliseconds.</p>

<p>So you want something like this</p>

<pre><code>public class MyFrame extends javax.swing.JFrame {

   private Timer timer = null;
   ImageIcon[] icons = new ImageIcon[3];
   int index = -1;

   public MyFrame() {
       initComponents();
       icons[0] = new ImageIcon(...);
       icons[1] = new ImageIcon(...);
       icons[2] = new ImageIcon(...);

       timer = new Timer(2000, new ActionListener(){
           public void actionPerformed(ActionEvent e) {
               if (index + 1 &gt; 2) {
                   index = 0;
                   jLabel3.setIcon(icons[index]);
               } else {
                   index++;
                   jLabel3.setIcon(icons[index]);
               }
           }
       });
   }
}
</code></pre>

<p>For your button, you need to use <code>mouseEntered</code> <em>and</em> <code>mouseExited</code>, then you can just call <code>timer.start()</code> or <code>timer.stop()</code></p>

<pre><code>private void jButton1MouseExited(MouseEvent e) {
    timer.stop();
}

private void jButton1MouseEntered(MouseEvent e) {
    timer.start();
}
</code></pre>

<p>If you don't know how to add the <code>MouseListener</code> just right click on the button from the design view and select <code>Event -&gt; Mouse -&gt; mouseEntered</code>. Do the same for <code>mouseExited</code>.  You should see the above methods auto-generated for you.</p>
