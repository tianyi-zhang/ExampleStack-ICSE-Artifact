<p>There are 2 changes need to your existing implementation, I didn't realized that before.</p>

<p>First of all, you need to look if you list in <code>WordContainer</code> has already got that word or not. If not then you need to add it.</p>

<p>And second, for that to work, you need to override equals method in your Word method as the <code>contains</code> method would try to compare objects using <code>equals</code></p>

<pre><code>public Word put(Word s) {
        if (al.contains(s)) {
            for (Word w : al) {
                if (w.s.compareTo(s.getWord()) == 0) {
                    s.setfrequency();
                }// else {           //Remove this else, as you add element outside
                 // al.add(s);
                 //}
            }
        } else
            al.add(s);
        return s;
    }
</code></pre>

<pre><code>    @Override
    public boolean equals(Object obj) {
        if (this == obj)
            return true;
        if (obj == null)
            return false;
        if (getClass() != obj.getClass())
            return false;
        Word other = (Word) obj;
        if (frequency != other.frequency)

            if (other.s != null)
