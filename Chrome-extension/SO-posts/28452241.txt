<p>This is how I have achieved it with JPanel visible property.</p>

<p><strong>Basic idea</strong></p>

<ol>
<li>Show frame of user login/register Set it's appropriate size say
(200,400) like Form,</li>
<li>Add two JPanels (login and register) in it with size (200,400), and hide any one say Register. </li>
<li>Your frame will be look like having only one form. </li>
<li>When user click Register button from login panel, you have to do two things </li>
</ol>

<p><strong>loginPanel.setVisible(false);</strong>
    and <strong>registerPanel.serVisible(true);</strong></p>
