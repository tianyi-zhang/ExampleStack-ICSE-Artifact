<p>A more Android way of doing it:</p>

<pre><code>    public static int manipulateColor(int color, float factor) {
        int a = Color.alpha(color);
        int r = Math.round(Color.red(color) * factor);
        int g = Math.round(Color.green(color) * factor);
        int b = Math.round(Color.blue(color) * factor);
        return Color.argb(a,
                Math.min(r,255),
                Math.min(g,255),
                Math.min(b,255));
    }
</code></pre>

<p>You will want to use a factor less than <code>1.0f</code> to darken. try <code>0.8f</code>.</p>
