<p>IMO, you have to add your own JPanel into ContentPane of JFrame.</p>

<p>Besides, use SwingUtilities to show GUI.</p>

<pre><code>    public static void main(String[] args) {
        SwingUtilities.invokeLater(new Runnable() {    
            public void run() {
                JFrame f = new JFrame();
                NewJPanel n1 = new NewJPanel();
                f.getContentPane().add(n1, BorderLayout.CENTER);
                f.pack();
                f.show();
            }
        });
    }
</code></pre>
