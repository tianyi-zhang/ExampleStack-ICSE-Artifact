This folder contains the artifact for ICSE Paper#401. There are three major components. 

The Datasets folder includes the adaptation dataset and variation dataset (Section II in the ICSE paper).

The Classification folder includes the manual inspection spreadsheet and the automated classification tool (Section III and Section IV in the ICSE paper).

The Chrome-extension folder includes the chrome extension, the back-end server, and a tool manual (Section V in the ICSE paper).

Please refer to the README file in each subfolder for detailed instructions.
